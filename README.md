# Smart Expense Tracker

## Overview

Smart Expense Tracker is a simple and responsive web application that helps users manage their daily expenses efficiently. The application allows users to record expenses, categorize spending, monitor financial activity, and visualize spending patterns through analytics.

The project is built using HTML, CSS, and JavaScript and stores data locally using the browser's LocalStorage feature. No backend server or external database is required.

---

## Features

### Dashboard

* View Total Expenses
* View Total Transactions
* View Highest Expense
* View Today's Expenses

### Expense Management

* Add new expenses
* Delete individual expenses
* Clear all expense records
* Form validation for accurate data entry

### Search and Filter

* Search expenses by name
* Filter expenses by category

### Analytics

* Category-wise spending analysis
* Visual spending chart

### Data Storage

* Persistent storage using LocalStorage
* Data remains available after page refresh

### Export Functionality

* Export expense records as CSV files

### User Experience

* Responsive design for desktop and mobile devices
* Dark Mode and Light Mode support
* Toast notifications for user actions
* Confirmation dialogs before deleting data

## How LocalStorage is Used

The application stores all expense records in the browser using LocalStorage.

Stored Data:

* Expense Name
* Amount
* Category
* Date
* Theme Preference

Benefits:

* No database required
* Data persists after browser refresh
* Fast and lightweight storage

---

## Installation and Usage

1. Download the project files.
2. Open `index.html` in any modern web browser.
3. Start adding expenses.
4. View analytics and dashboard statistics.
5. Export records whenever needed.

No installation, server setup, or database configuration is required.
